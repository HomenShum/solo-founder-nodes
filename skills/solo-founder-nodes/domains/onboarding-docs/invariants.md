# Onboarding Docs Domain Pack

Jobs:

- turn a model into a teaching artifact,
- bind hotspots to real components,
- show an ordered assembly/process sequence,
- ask coach/quiz questions tied to source notes.

Blocking gates:

- hotspot-component binding proof,
- sequence proof,
- coach-review proof.

This domain is not judged by mesh fidelity alone; it is judged by teachability.
